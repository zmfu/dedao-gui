### 编译
1.windows
```
cd dd-vue
npm i
cd ../dd-server
npm i
npm run dist:win
```
2.mac
```
cd dd-vue
npm i
cd ../dd-server
npm i
sudo npm run dist:macarm 或 sudo npm run dist:macx64
```
